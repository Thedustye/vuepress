# 玩mc相比会有许多疑难问题，这个是解答

# 关于进服问题

Can't resolve hostname 服务器IP输入错误，或者目前网络状态极差

Connection refused: connect 寻找不到服务器，或者服务器关闭中，建议检测IP是否正确

Connection Timed Out 服务器或者你的网络极差，或者服务器正在执行大型数据计算，例如使用Worldedit插件编辑大量方块时

Failed to login: User not premium 或 Bad login 盗版尝试登录正版服务器，可以在Server.properties内关闭正版验证

Outdated server ! 服务器版本过旧（根据你目前使用的客户端而言）

Outdated client ! 服务器版本过新（根据你目前使用的客户端而言）

java.net.SocketException: Connection reset by peer […] 你的网络状态极差或者即将掉线——正版服务器有可能是因为Minecraft.net验证服务器出现错误

You are not white-listed on this server 你不在服务器白名单内

You are banned from this server ! 你被Ban了

Internal server error 服务器数据错误（原因很多可以是插件导致/Mod导致等…）

End of Stream 你的网络状态极差，或者服务器取读不了客户端内的一部分数据，通常有安装MOD的客户端才会出现次症状，也有可能短时间内频繁登录服务器。

Internal Exception: io.netty.handler.timeout.ReadTimeoutException 客户端取读不了服务器传输的数据包（过大），或者服务器状态不佳

java.io.IOException: Server returned HTTP response code: 503 正版服务器验证服务器掉线，导致验证失败

Took to long to log in 同上，或者登录花时过长稍后再试

Server closed 服务器关闭。

Your country is banned from this server Authme开启了国家黑名单保护，用N++打开Authme文件夹里config.yml里面的181行给删除了就可以了

You have connected too fast. You must wait at least 4 seconds between connections. 这个是在连接BC服时，你连接太快了，需要等4秒才能重新连接

The server you were previously on went down, you have been connected to the lobby 这个是BC服插件的问题，详细请查看BC后台的报错信息

Server is full! 服务器已经满人！

Could not connect to default or fallback server, please try again later: [...] 连接BC服不能连接，这个提示是因为大厅没有开。或者是其他的原因

You are already connected to this server! 你已经连接到了这个服务器。

Already connecting to this server! 正在连接这个服务器中。

# 其他问题

Q:为什么注册/登录账号时一直不成功？

注册/登录的格式不对。注册格式：/register 密码 第二遍密码 登录格式：/login 密码

例：密码为114514，则第一次进游戏注册输入/register 114514 114514

之后再次进游戏登录输入/login 114514

Q:如何创建领地？

使用木锄左键/右键选点，然后输入/res create 领地名

Q：为什么我tpa其他玩家，他接受之后我还是没有传送过去？

对方可能在你没有传送权限的领地内，走出领地或输入/res set tp true将传送权限打开即可
